TP CORRESPONDIENTE A RECURSIVIDAD CON 8 EJERCICIOS PRÁCTICOS
Este trabajo consiste en la implementación de funciones recursivas en Python para resolver diversos problemas matemáticos y computacionales. 
Incluye cálculos como el factorial, la serie de Fibonacci, la potencia de un número, conversiones numéricas, y análisis de cadenas y dígitos. 
Además, incorpora un programa principal que permite al usuario interactuar con cada función mediante la entrada de datos.
